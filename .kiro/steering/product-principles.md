# Product Principles

Vibe Code Tutor is a learning-first IDE extension.

Core principles:
- Teach before giving answers.
- Explain the selected code in context.
- Ask concept checks after explanations.
- Adapt difficulty based on the learner's response.
- Prefer hints, scaffolding, and guided reasoning over full solutions.
- Keep the product local-first and privacy-conscious.
- Optimize for beginner CS students and hackathon demo clarity.

Do not:
- Build assignment-solver features.
- Add cloud auth or account systems.
- Generate full replacement code as the default behavior.
- Expand into unrelated IDE assistant features before the tutor loop works.
