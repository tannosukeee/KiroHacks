# Project Overview

## Product name
Vybe Tutor

## Product definition
Vybe Tutor is a local-first AI coding tutor built as a VS Code / Kiro-style IDE extension. It helps students understand code they write, paste, or accept from AI tools by showing contextual explanations, comprehension checks, and lightweight progress feedback directly in the editor sidebar.

## Core problem
AI-assisted coding helps students produce code faster, but it can weaken conceptual understanding when students accept generated code without knowing how or why it works. Vybe Tutor closes that gap by turning passive vibe coding into active learning.

## Primary users
- Beginner CS students learning programming fundamentals
- Intermediate CS students working with unfamiliar code, frameworks, or patterns
- Self-taught developers who want structured explanations tied to their own code
- Hackathon participants who need fast, low-pressure comprehension support

## Core value proposition
- Explain code in context instead of sending users to a separate learning platform.
- Teach concepts before giving answers.
- Ask short comprehension checks after explanations.
- Adapt question difficulty based on the learner's answer history.
- Reward learning behavior through XP, levels, and streaks.
- Keep user progress and preferences local whenever possible.

## Judging alignment
- Implementation: real IDE extension, Gemini integration, structured AI outputs, Zod validation, deterministic adaptive engine, local persistence.
- Innovation & Design: learning layer inside the coding workflow, polished sidebar experience, gamified motivation, adaptive comprehension checks.
- Social Good: supports students who use AI but need accessible, confidence-building programming education.

## Naming rule
Use `Vybe Tutor` in user-facing copy. Avoid older names such as `Vibe Code Tutor` unless referring to legacy docs.
