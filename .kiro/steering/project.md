# Product Overview

We are building a local-first AI coding tutor as a VS Code / Kiro-style IDE extension.

The extension helps beginner programmers understand their own code, receive concept-based explanations, and practice with adaptive quiz questions.

Primary users:
- Beginner CS students
- Hackathon participants
- Students who feel lost reading compiler errors or unfamiliar code

Core value:
- Do not just give answers.
- Teach the concept.
- Generate follow-up practice.
- Adapt difficulty based on the learner’s response.

Judging alignment:
- Implementation: real IDE extension, Gemini integration, adaptive difficulty engine, local persistence.
- Innovation & Design: tutoring directly inside the editor, minimal workflow disruption, polished learner experience.
- Social Good: supports students who need accessible, low-pressure programming help.
