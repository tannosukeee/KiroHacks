# Project Overview

## Product name
Vybe Tutor

## Product definition
Vybe Tutor is a local-first AI coding tutor built as a VS Code / Kiro-style IDE extension. It helps students understand code they write, paste, or accept from AI tools by showing contextual explanations, comprehension checks, gamified learning feedback, and optional course-material connections directly in the IDE sidebar.

## Core problem
AI-assisted coding helps students produce code faster, but it can weaken conceptual understanding when students accept generated code without knowing how or why it works. Vybe Tutor closes that gap by turning passive vibe coding into active learning.

## Primary users
- Beginners who are learning syntax, logic, and foundational CS concepts for the first time.
- Intermediate students who can write functional code but want help understanding larger codebases, frameworks, and design choices.
- Advanced learners or early-career developers who want concise explanations of architecture, patterns, optimization, and best practices.
- Hackathon participants who need fast, low-pressure comprehension support without leaving their editor.

## Core value proposition
- Explain code in context instead of sending users to a separate learning platform.
- Teach concepts before giving direct answers.
- Ask short comprehension checks after explanations.
- Adapt question difficulty and explanation depth based on learner performance.
- Reward learning behavior through XP, levels, and daily streaks.
- Support optional imported course materials so explanations can connect to syllabi or lecture notes.
- Keep user progress, preferences, and imported-material summaries local whenever possible.

## Judging alignment
- Implementation: real IDE extension, Gemini integration, structured AI outputs, Zod validation, deterministic adaptive engine, gamification, local persistence.
- Innovation & Design: learning layer inside the coding workflow, polished sidebar experience, adaptive comprehension checks, deep-dive follow-up questions, and gamified motivation.
- Social Good: supports students who use AI but need accessible, confidence-building programming education.

## Naming rule
Use `Vybe Tutor` in user-facing copy. Avoid older names such as `Vibe Code Tutor` or `Vibe Code Tutor` unless referring to legacy docs.
